# Human First Systems Portfolio Demos

Static portfolio demos for AI workflow, revenue operations, job-search, and production systems.

## Main demos

- `ai-revenue-operator.html` - AI Revenue Operator portfolio demo for AI, GTM, RevOps, and operator roles.
- `job-hunter.html` - Job Hunter AI Agent Builder for tailored job-search assets.
- `index.html` - Human First Systems workflow diagnostic.

## Publish

This project is static HTML, CSS, and JavaScript. It can be hosted with GitHub Pages, Netlify, Vercel, or any static host.

For GitHub Pages, publish the repo from the `main` branch and set Pages to serve from the repository root.
